Detailed Overview of CSS

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation (look and feel) of web pages written in HTML or XML. It defines how elements should appear—layout, colors, fonts, spacing, and animations—across different devices and screen sizes.

Key Features of CSS

  • Separation of Content & Style: HTML handles structure/content, while CSS controls design/appearance.
  • Cascading Nature: CSS follows a cascade hierarchy where the last applied or most specific rule takes priority.
  • Reusability: A single CSS file can style multiple HTML pages.
  • Responsive Design: Supports media queries to adapt layouts for different screen sizes.
  • Cross-Browser Compatibility: Works on all modern browsers.

Types of CSS

  • Inline: Applied directly to an HTML element using the style attribute.
  • Internal: Written inside a <style> tag in the <head> section of an HTML file.
  • External: Stored in a separate .css file and linked via a <link> tag.

CSS Syntax

The basic syntax is:

selector {
    property: value;
}

Types of Selectors

  • Universal Selector: * { margin: 0; }
  • Type Selector: p { color: black; }
  • Class Selector: .highlight { background: yellow; }
  • ID Selector: #header { text-align: center; }
  • Attribute Selector: input[type="text"] { border: 1px solid gray; }
  • Pseudo-classes: a:hover { color: red; }
  • Pseudo-elements: p::first-letter { font-size: 2em; }

Core CSS Properties

  • Text & Font: color, font-family, font-size, line-height
  • Backgrounds: background-color, background-image
  • Box Model: margin, padding, border, width, height
  • Positioning: position, top, left, z-index
  • Layout: display, flexbox, grid
  • Animations & Transitions: transition, transform, @keyframes

CSS Box Model

Every HTML element is treated as a rectangular box with:

  • Content: Text or images inside the element.
  • Padding: Space between content and border.
  • Border: Surrounds the padding.
  • Margin: Space outside the border.

Responsive Design

CSS helps create layouts that adapt to different devices using:

  • Media Queries
  • Flexbox & Grid for fluid, dynamic layouts

CSS Preprocessors

Advanced tools like Sass/SCSS, LESS, and Stylus add variables, nesting, mixins, and functions to simplify CSS management.

Modern CSS Features

  • CSS Variables (Custom Properties)
  • CSS Grid Layout
  • Flexbox
  • Transitions & Animations
  • Dark Mode Support

Advantages of CSS

  • Better performance (lighter pages)
  • Easier maintenance and scalability
  • More control over presentation
  • Ensures cross-device responsiveness
